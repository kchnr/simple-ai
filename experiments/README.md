# Experiments Project

This directory contains all Python experiments for rapid prototyping and validation of neural plasticity, learning rules, and evolutionary mechanisms.

## Setup (using `uv`)

We use [`uv`](https://github.com/astral-sh/uv) for fast, reproducible Python package management. To get started:

1. **Install uv** (if not already):
   ```sh
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```

2. **Create a virtual environment:**
   ```sh
   uv venv .venv --python=3.11
   ```

3. **Install dependencies:**
   ```sh
   uv pip install -r requirements.txt
   ```

4. **Run experiments:**
   ```sh
   .venv/bin/python main.py
   ```

## Project Structure
- `main.py` — Orchestrates experiments
- `network.py` — Hebbian network implementation
- `tasks.py` — Task/data generation
- `utils.py` — Utility functions
- `requirements.txt` — Python dependencies
- `requirements.lock` — Locked dependency versions (generated by uv)
- `experiment_readme_template.md` — Template for documenting experiments

## Notes
- All experiments share this environment and codebase.
- Use the template to document new experiments.
- For reproducibility, always update `requirements.lock` after changing dependencies:
  ```sh
  uv pip compile requirements.txt -o requirements.lock
  ```
